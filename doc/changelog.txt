0.12.0.2013-12-06
Erweiterung:	Pokalwertung (cupscoring.php)
		Ein Stand nach x von y Wettbewerben wird automatisch vom System erzeugt. Dazu sind die betreffenden PokalGrxx.c01-Dateien bei Aktualisierung aus dem
		COSAWIN-Verzeichnis derzeit noch mit der Hand zu kopieren, da keine automatische Kopie erfolgt.
		Links zu den Wertungen werden über dem Zeitplan angezeigt.
		Zudem ist es möglich, Wertungen kombiniert auszugeben.
		Bsp.:	Als einzelne Pokalwertungen sind vorhanden: (01) Männer, (02) Frauen.
			Dies Punkte beider Wertungen sollen zusammenaddiert werden und in der Wertung Gesamt ausgegeben werden.
		In der Zip-Datei mit den LaIVE-Dateien liegt eine Datei: "beispiel_laive_combinedcupscoring.txt", soll die kombinierte Wertung genutzt werden, muss
		diese in "laive_combinedcupscoring.txt" umbenannt werden (ohne beispiel_).

		Zum Aufbau der Datei:
		Es handelt sich um eine trennzeichenseperierte Datei. Trennzeichen: ;
		Folgende Felder sind vorhanden
		1) Nummer der kombinierten Wertung (frei wählbar; bei einstellig, muss eine 0 vor die Ziffer)
		2) Name der kombinierten Wertung
		3) Nummern, der Wertungen, die zusammengeführt werden sollen, getrennt durch Komma (Zahlen im Format wie bei 1))

		Beispielinhalt der Datei für das obige Beispiel:
		01;Gesamt;01,02;
		
		Ist diese Datei auf dem Server mit den Angaben, wird automatisch die kombinierte Pokalwertung berechnet und auch angezeigt (Link ebenfalls über Zeitplan)

		Zudem ist es möglich, für kombinierte Pokalwertungen die Ergebnisse als trennzeichenseperierte Datei auszugeben, damit diese z. B. in ein Word-Dokument
		(Serienbrieffunktion) zum Urkundendruck eingebunden werden können. Dazu ist die Seite mit folgender ergänzender Angabe in der URL auszurufen: cccsv=1
		(Der Link hierzu erscheint unten.

0.5.0.2013-05-21

Liste der benötigten Dateien wurde im Modul Startlistenerstellen  um Verein.c01  und Stamm.c01 ergänzt.

Erweiterungen:
Neu: Administrationsmodul Startlisten erstellen (startlistenerstellen.php): 
     - Aufruf mit: index.php?sub=startlistenerstellen.php.
     - Alle 5 Minuten wird der Aufruf durch ein Refresh automatisch durchgeführt. Durch manuelles Aufrufen der Seite ist dies auch manuell steuerbar.
     - Das Modul ist in der Lage aus COSAWIN Startlisten zu erstellen und diese als HTML-Datei auf dem Server anzulegen, so dass sie danach sofort zur Ansicht bereit-
       stehen und in der Übersicht und im Zeitplan angezeigt werden.
     - Es werden derzeit berücksichtigt: Startlisten Einzel und Startlisten Staffel.
     - Qualifikationsmodus wird mit ausgegeben bei Vorrunden.
     - die Meldeleistungen oder Leistungen aus Vorrunden werden berücksichtigt und mit ausgegeben.
     - Gemischte Wettbewerbe: auch hier werden die Startlisten erstellt, die entsprechende Altersklassen wird hinter jeden Sportler ausgegeben.
     - Zudem wird die Startliste bei gemischten Wettbewerben für jede beteiligte Altersklasse erstellt.

0.1.0.2013-05-07
Das System nutzt COSAWIN-Veranstaltungsdateien und HTML-Ausgaben aus COSAWIN mittels des Result-Tickers um diese auf dem Server entsprechend aufbereitet zu präsentieren
Grundsystem mit folgenden Modulen ist vorhanden:
1) Übersicht:	- Teilnehmerlisten und Ergebnislisten werden angezeigt, sobald diese auf dem Server vorhanden sind. Die aktuellsten sind jeweils oben zu finden.
				- Tabelle ist nach jeder Spalte durch Klick auf den jweiligen Tabellenkopf sortierbar.
				- Übersicht wird alle 2 Minuten automatisch aktualisiert.
2) Zeitplan:	- Zeitplan wird von COSAWIN eingelesen (Datei: Wettbew.c01). Diese muss auf dem Server vorhanden sein; hin kopiert
                - Zeitplan wird automatisch generiert und zeigt die Wettbewerbe an.
				- Zeitplanseite wird alle 5 Minuten automatisch aktualisiert.
3) Allgemein:	- Überschriften und Datum/Veranst.-Tage werden aus COSAWIN übernommen. Dazu muss die Datei vandat.c01 auf den Server kopiert werden.
				- Folgende Typen/Runden werden derzeit unterstützt (für Detais siehe Zusammenstellungen über die Nummern und Dateiformate: )
					Typen: 	ohne (es liegen nur Zeitplandaten vor), Teilnehmer, Ergebnisse: Vorläufe, Zeitvorläufe, Zwischenläufe, Finale (letzte Runde, egal was),
							Zwischenergebnisse (bei Finals), Zwischenergebnisse bei Mehrkämpfen nach jeweiliger Disziplin
					Runden:	Vorläufe, Zeitvorläufe, Zwischenläufe, Finale, nach x Disziplinen (beim Mehrkampf)
Über das System: 
- Es besteht aus in PHP geschriebenen Seiten. Getestet wurde es auf einem Apache-Server mit PHP 5.4.
